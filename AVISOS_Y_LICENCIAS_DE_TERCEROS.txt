AVISOS Y LICENCIAS DE COMPONENTES DE TERCEROS
RoboScript 1.7.1
Actualizado: 23 de julio de 2026

RoboScript incorpora software de terceros. Cada componente conserva sus propios derechos de autor y condiciones de licencia. La licencia propia de RoboScript no modifica ni restringe los derechos otorgados por esas licencias.

1. QT FOR PYTHON / PYSIDE6 / SHIBOKEN6

Componentes utilizados:
- PySide6 6.10.2
- PySide6 Essentials 6.10.2
- PySide6 Addons 6.10.2
- Shiboken6 6.10.2

Qt for Python se distribuye bajo la GNU Lesser General Public License versión 3 (LGPLv3), la GNU General Public License versión 3 (GPLv3) y una licencia comercial de Qt, según corresponda.

RoboScript utiliza la edición comunitaria bajo LGPLv3 exclusivamente para aquellos módulos disponibles bajo esa licencia.

Conforme a LGPLv3:
- el usuario conserva los derechos que la LGPLv3 otorga sobre las bibliotecas cubiertas;
- no se restringe la modificación o sustitución de esas bibliotecas;
- no se restringe la ingeniería inversa cuando sea necesaria para depurar modificaciones realizadas en dichas bibliotecas;
- las bibliotecas cubiertas permanecen separadas dentro de la distribución portable;
- se proporcionan los textos completos de LGPLv3 y GPLv3 junto con la distribución;
- se proporcionan instrucciones para identificar y obtener el código fuente correspondiente de las versiones utilizadas.

3. PYINSTALLER

RoboScript fue empaquetado utilizando PyInstaller 6.19.0.

PyInstaller utiliza un esquema de licencia dual que incluye GPL 2.0 con una excepción que permite usar el programa para construir y distribuir aplicaciones comerciales o propietarias. Algunos archivos se distribuyen bajo Apache License 2.0.

Licencia oficial:
https://pyinstaller.org/en/stable/license.html

PyInstaller es una herramienta de construcción. El uso de PyInstaller no convierte automáticamente RoboScript en software GPL.

4. OTROS COMPONENTES

La distribución puede incluir componentes transitivos utilizados por Python, Qt o PyInstaller. Sus avisos de licencia deben conservarse en la carpeta de licencias incluida en el paquete portable.

Para Qt y Qt for Python, solo se incluyen y reconocen los módulos y componentes realmente distribuidos con RoboScript.

6. CÓDIGO FUENTE DE COMPONENTES LGPL

El código fuente correspondiente de las bibliotecas LGPL utilizadas por RoboScript puede obtenerse desde los sitios oficiales indicados anteriormente.

Durante al menos el periodo exigido por LGPLv3, el proyecto El Origen mantendrá instrucciones públicas suficientes para identificar y obtener las fuentes correspondientes a la versión exacta distribuida.

7. AUSENCIA DE MODIFICACIONES

A la fecha de esta versión, el proyecto El Origen no declara modificaciones propias al código fuente de Qt, PySide6 o Shiboken6. Si en una versión futura se realizan modificaciones, deberán publicarse las fuentes correspondientes conforme a la licencia aplicable.

Este documento es informativo y no sustituye los textos completos de las licencias aplicables.
